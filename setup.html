Installing the Java Development Kit (JDK)

The Java Development Kit (JDK) is a software development environment used for developing Java applications. It includes the Java Runtime Environment (JRE), an interpreter/loader (Java), a compiler (javac), an archiver (jar), a documentation generator (Javadoc), and other tools needed for Java development.

Steps to Install the JDK:

1

Download the JDK

Visit the Oracle JDK download page or AdoptOpenJDK for an open-source alternative.

2

Select Your Operating System

Choose the JDK version appropriate for your operating system (Windows, macOS, or Linux).

3

Run the Installer

Execute the downloaded installer and follow the installation instructions.

4

Verify Installation

Open a terminal or command prompt and type java -version to verify that Java is installed correctly.

Setting Up IntelliJ IDEA

IntelliJ IDEA is one of the most popular Java IDEs. Here's how to set it up:

1

Download IntelliJ IDEA

Visit the JetBrains website and download either the Community (free) or Ultimate (paid) edition.

2

Install the IDE

Run the installer and follow the installation wizard.

3

Launch and Configure

Open IntelliJ IDEA and configure it with your preferred settings (theme, keymap, etc.).

4

Set Up the JDK

Go to File > Project Structure > SDKs and add your installed JDK if it's not automatically detected.

Tip

IntelliJ IDEA offers a wide range of plugins. Consider installing the "Checkstyle" plugin for code quality checks and "SonarLint" for static code analysis.

Setting Up Eclipse

Eclipse is a free, open-source IDE that's popular among Java developers. Here's how to set it up:

1

Download Eclipse

Visit the Eclipse download page and download the Eclipse IDE for Java Developers.

2

Extract the Archive

Unlike many other programs, Eclipse doesn't have an installer. Simply extract the downloaded archive to your preferred location.

3

Launch Eclipse

Run the eclipse.exe (Windows) or Eclipse application (macOS) file.

4

Configure the Workspace

Choose a workspace directory when prompted. This is where your projects will be stored.

5

Set Up the JDK

Go to Window > Preferences > Java > Installed JREs and add your installed JDK if it's not already there.

Configuring Environment Variables

Setting environment variables allows you to run Java commands from any directory in the command prompt or terminal.

Windows
macOS
Linux

Setting PATH on Windows

  1. Right-click on "This PC" or "My Computer" and select "Properties"
  2. Click on "Advanced system settings"
  3. Click on "Environment Variables"
  4. Under "System variables", find and select "Path", then click "Edit"
  5. Click "New" and add the path to the JDK bin folder (e.g., C:\Program Files\Java\jdk-17\bin)
  6. Click "OK" to close all dialogs

Setting PATH on macOS

  1. Open Terminal
  2. Open your shell profile file: nano ~/.bash_profile or nano ~/.zshrc
  3. Add the following line: export PATH=$PATH:/Library/Java/JavaVirtualMachines/jdk-17.jdk/Contents/Home/bin
  4. Press Ctrl+O to save, then Ctrl+X to exit
  5. Reload the profile: source ~/.bash_profile or source ~/.zshrc

Setting PATH on Linux

  1. Open Terminal
  2. Open your shell profile file: nano ~/.bashrc
  3. Add the following line: export PATH=$PATH:/usr/lib/jvm/jdk-17/bin
  4. Press Ctrl+O to save, then Ctrl+X to exit
  5. Reload the profile: source ~/.bashrc

Verifying Your Setup

To verify that your environment variables are set correctly, open a new terminal or command prompt window and run:

Command 
java -version
javac -version

Both commands should display the version of the JDK you installed.

Key Takeaways

  • The JDK (Java Development Kit) is essential for Java development and includes tools like the compiler (javac) and JRE.
  • Popular Java IDEs include IntelliJ IDEA, Eclipse, NetBeans, and Visual Studio Code with extensions.
  • Setting up environment variables allows you to run Java commands from any directory.
  • It's important to verify your installation by checking Java and Javac versions in the terminal.
  • Choose an IDE that fits your needs based on your experience level and project requirements.
