The "Hello, World!" Program

The "Hello, World!" program is traditionally the first program that programmers write when learning a new language. It's a simple program that outputs the text "Hello, World!" to the screen.

Here's what a basic Hello World program looks like in Java:

Java 
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Let's save this code in a file named HelloWorld.java. The filename must match the class name exactly, including capitalization.

Understanding the Program Structure

Let's break down the "Hello, World!" program to understand its structure:

public class HelloWorld {

This line declares a class named HelloWorld. In Java, everything must be inside a class. The public keyword means this class is accessible from other classes.

public static void main(String[] args) {

This is the main method, which is the entry point for Java applications. When you run a Java program, it starts executing from this method.

  • public: This method is accessible from outside the class.
  • static: This method belongs to the class rather than an instance of the class.
  • void: This method doesn't return any value.
  • main: The name of the method, which must be exactly "main" for the entry point.
  • String[] args: The method accepts an array of strings as a parameter, which can contain command-line arguments.
System.out.println("Hello, World!");

This line displays the text "Hello, World!" to the console.

  • System: A class that contains useful members related to the system.
  • out: A static member of the System class, representing the standard output stream.
  • println(): A method of the PrintStream class (which out is an instance of) that prints a line of text to the console.
}

This closing brace marks the end of the main method.

}

This closing brace marks the end of the HelloWorld class.

Compiling and Running Your Java Program

To run a Java program, you need to first compile it and then execute it. Here are the steps:

1

Create the Java File

Save the code in a file named HelloWorld.java. Remember, the filename must match the class name.

2

Open Command Prompt or Terminal

Navigate to the directory where you saved the file using the cd command.

3

Compile the Program

Type javac HelloWorld.java and press Enter. This will compile your Java source code into bytecode (a .class file).

4

Run the Program

Type java HelloWorld (note: no .java extension) and press Enter. This will execute your program.

Note

Notice that we use javac to compile the program and java to run it. The javac command creates a file named HelloWorld.class, which contains the bytecode that the Java Virtual Machine (JVM) will execute.

Using an IDE

If you're using an IDE like IntelliJ IDEA or Eclipse, the process is simpler:

  1. Create a new Java project
  2. Create a new Java class named HelloWorld
  3. Enter the code in the editor
  4. Click the Run button or press Ctrl+Shift+F10 (IntelliJ) or Ctrl+F11 (Eclipse)

Common Errors for Beginners

Here are some common errors that beginners might encounter when writing their first Java program:

Incorrect Capitalization

Java is case-sensitive. helloworld, HelloWorld, and HELLOWORLD are all different identifiers in Java.

Java 
public class helloworld {  // Error! File name should be helloworld.java
    public static void Main(String[] args) {  // Error! Method name should be main, not Main

Missing Semicolons

In Java, statements must end with a semicolon (;).

Java 
System.out.println("Hello, World!")  // Error! Missing semicolon

Mismatched Braces

Every opening brace { must have a corresponding closing brace }.

Java 
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    // Error! Missing closing brace for the class

Incorrect File Name

The file name must match the class name exactly, including capitalization, and have a .java extension.

For example, the class HelloWorld should be in a file named HelloWorld.java, not helloworld.java or HelloWorld.txt.

Key Takeaways

  • A Java program must be written inside a class.
  • The main method is the entry point for Java applications.
  • Java is case-sensitive, so be careful with capitalization.
  • The file name must match the class name and have a .java extension.
  • Use javac to compile and java to run your program.
  • Common errors include missing semicolons, mismatched braces, and incorrect capitalization.
