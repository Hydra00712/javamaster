Variables and Data Types

In Java, a variable is a container that holds a value during the execution of a program. Before using a variable, you need to declare it with a specific data type.

Primitive Data Types

Java has eight primitive data types:

Data Type Size Range Default Value Example
byte 8 bits -128 to 127 0 byte b = 10;
short 16 bits -32,768 to 32,767 0 short s = 1000;
int 32 bits -2^31 to 2^31-1 0 int i = 100000;
long 64 bits -2^63 to 2^63-1 0L long l = 100000L;
float 32 bits ±3.4e−038 to ±3.4e+038 0.0f float f = 3.14f;
double 64 bits ±1.7e−308 to ±1.7e+308 0.0d double d = 3.14159;
char 16 bits 0 to 65,535 '\u0000' char c = 'A';
boolean 1 bit true or false false boolean b = true;

Reference Data Types

Reference data types are used to access objects. Examples include:

  • Classes (e.g., String, Scanner)
  • Interfaces
  • Arrays
Java 
// Primitive data types
int age = 25;
double salary = 50000.50;
char grade = 'A';
boolean isStudent = true;

// Reference data types
String name = "John Doe";
int[] numbers = {1, 2, 3, 4, 5};

Operators

Java provides a rich set of operators for manipulating variables. Here are the main categories:

Arithmetic Operators

Operator Description Example
+ Addition int sum = a + b;
- Subtraction int diff = a - b;
* Multiplication int product = a * b;
/ Division int quotient = a / b;
% Modulus (remainder) int remainder = a % b;
++ Increment a++; or ++a;
-- Decrement a--; or --a;

Comparison Operators

Operator Description Example
== Equal to if (a == b) {...}
!= Not equal to if (a != b) {...}
> Greater than if (a > b) {...}
< Less than if (a < b) {...}
>= Greater than or equal to if (a >= b) {...}
<= Less than or equal to if (a <= b) {...}

Logical Operators

Operator Description Example
&& Logical AND if (a > 0 && b > 0) {...}
|| Logical OR if (a > 0 || b > 0) {...}
! Logical NOT if (!isTrue) {...}

Assignment Operators

Operator Description Example Equivalent to
= Simple assignment a = 5; N/A
+= Add and assign a += 5; a = a + 5;
-= Subtract and assign a -= 5; a = a - 5;
*= Multiply and assign a *= 5; a = a * 5;
/= Divide and assign a /= 5; a = a / 5;
%= Modulus and assign a %= 5; a = a % 5;

Control Flow Statements

Control flow statements are used to control the execution flow of a program.

Conditional Statements

Conditional statements allow you to execute different blocks of code based on different conditions.

If Statement

Java 
if (condition) {
    // code to be executed if condition is true
}

If-Else Statement

Java 
if (condition) {
    // code to be executed if condition is true
} else {
    // code to be executed if condition is false
}

If-Else-If Ladder

Java 
if (condition1) {
    // code to be executed if condition1 is true
} else if (condition2) {
    // code to be executed if condition2 is true
} else {
    // code to be executed if both conditions are false
}

Switch Statement

Java 
switch (expression) {
    case value1:
        // code to be executed if expression equals value1
        break;
    case value2:
        // code to be executed if expression equals value2
        break;
    default:
        // code to be executed if expression doesn't match any case
}

Looping Statements

Looping statements allow you to execute a block of code repeatedly.

For Loop

Java 
for (initialization; condition; update) {
    // code to be executed
}

// Example
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

While Loop

Java 
while (condition) {
    // code to be executed
}

// Example
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}

Do-While Loop

Java 
do {
    // code to be executed
} while (condition);

// Example
int i = 0;
do {
    System.out.println(i);
    i++;
} while (i < 5);

Jump Statements

Jump statements are used to transfer control to another part of the program.

Java 
// break - terminates the loop or switch statement
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        break; // exits the loop when i equals 5
    }
    System.out.println(i);
}

// continue - skips the current iteration and continues with the next
for (int i = 0; i < 10; i++) {
    if (i == 5) {
        continue; // skips printing when i equals 5
    }
    System.out.println(i);
}

// return - exits from a method
public int add(int a, int b) {
    return a + b; // returns the sum and exits the method
}

Arrays

An array is a container object that holds a fixed number of values of a single type. The length of an array is determined when it is created and cannot be changed.

Declaring and Creating Arrays

Java 
// Declaration
int[] numbers; // preferred syntax
int numbers[]; // alternative syntax

// Creation
numbers = new int[5]; // creates an array that can hold 5 integers

// Declaration and creation in one step
int[] numbers = new int[5];

// Declaration, creation, and initialization in one step
int[] numbers = {1, 2, 3, 4, 5};

Accessing Array Elements

Java 
int[] numbers = {1, 2, 3, 4, 5};

// Accessing elements (index starts from 0)
int firstElement = numbers[0]; // 1
int thirdElement = numbers[2]; // 3

// Modifying elements
numbers[1] = 10; // changes the second element to 10

// Array length
int length = numbers.length; // 5

Iterating Through Arrays

Java 
int[] numbers = {1, 2, 3, 4, 5};

// Using for loop
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// Using enhanced for loop (foreach)
for (int number : numbers) {
    System.out.println(number);
}

Multidimensional Arrays

Java 
// 2D array declaration and creation
int[][] matrix = new int[3][3];

// 2D array initialization
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Accessing elements in a 2D array
int element = matrix[1][2]; // row 1, column 2 (value: 6)

// Iterating through a 2D array
for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        System.out.print(matrix[i][j] + " ");
    }
    System.out.println();
}

Naming Conventions

Java has established naming conventions that enhance code readability and maintainability.

Element Convention Example
Classes Start with uppercase, use camelCase MyClass, StudentInfo
Methods Start with lowercase, use camelCase calculateTotal(), getName()
Variables Start with lowercase, use camelCase firstName, totalAmount
Constants All uppercase, words separated by underscore MAX_VALUE, PI
Packages All lowercase, domain name reversed com.example.project

Good Practices

  • Choose meaningful and descriptive names.
  • Avoid using abbreviations that aren't widely understood.
  • Keep names reasonably short but descriptive.
  • Don't use reserved keywords as names.

Key Takeaways

  • Java has eight primitive data types: byte, short, int, long, float, double, char, and boolean.
  • Reference data types are used to access objects such as classes, interfaces, and arrays.
  • Java provides various operators: arithmetic, comparison, logical, and assignment.
  • Control flow statements include conditional statements (if, if-else, switch) and loops (for, while, do-while).
  • Arrays are fixed-size containers that hold values of the same type, with index starting from 0.
  • Following Java naming conventions improves code readability and maintainability.
